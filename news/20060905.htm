pjsip.org News: Sept 5th, 2006

Added multi-purpose tone generator into PJMEDIA (tonegen.[hc]). Some of the features:

  • can generate single sinusoidal/sine wave or dual frequency tones,
  • fixed point or floating point,
  • can enqueue multiple tones/DTMF digits with single API call,
  • configurable on/off duration and volume for each individual tones/DTMF digits,
  • programmable digit-map, with default digit mapping set to DTMF frequencies.

You can view the public API in the tonegen.h - tone generator header file source code.

For other information, please visit pjsip.org - Open Source SIP Stack and Media Stack homepage
