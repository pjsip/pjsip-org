pjsip.org News: Sept 26th, 2006

Released version 0.5.8!

This release adds new features such as iLBC codec, experimental echo cancellation/AEC, Windows Mobile support, PUBLISH method, call transfer with norefersub support, and tone generator, as well as tons of bug fixes.

Please see the Release Notes for details, or go to PJSIP Download Page to get the files.
