pjsip.org News: Oct 21th, 2006

Audio on MacOS X fix

A dear mailing list member has reported several problems with audio on MacOS X 10.4, and subsequently this has been fixed few days back. Here is the discussions on the mailing list: http://www.pjsip.org/pipermail/pjsip/2006-October/000865.html

 

MP3 Call Recording

An MP3 file writer/recorder media port has been recently added to PJMEDIA. The media port converts a PCM input into MP3 file, and with this, it enables neat features such as call recording (including conference) into .MP3 files! (Thanks Toni for the contribution)

 

SIP call Transfer Fix

The SIP call transfer functionality in PJSUA was not really clever, as one mailing list member pointed out. It sent BYE request right after REFER is sent, and will cause REFER to fail if it is challenged because the remote INVITE session/dialog has been terminated.

This has been fixed by adding callback in PJSUA-LIB to report call transfer progress. Now applications such as PJSUA can monitor the transfer progress, and should only disconnect after the call transfer has completed successfully.
